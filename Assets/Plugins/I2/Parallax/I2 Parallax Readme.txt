----------------------
  Installation
----------------------

 1- Import the plugin package into a Unity project.
	
 2- Separate your backgrounds into Layers
 
 3- Add one of the I2Parallax_XXXX layers
	(more details: http://inter-illusion.com/assets/I2ParallaxManual/Layers.html)

The documentation provides further explanation on each of those steps and some tutorials.
Also its presented how to convert an existing UI into the I2 TextAnimation system.
						   (http://inter-illusion.com/assets/I2ParallaxManual/QuickStart.html)
